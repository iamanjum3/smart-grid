# ⚡ Smart Grid DSM Platform

## 📌 Project Description
This project simulates a Cloud-Based Smart Grid Demand-Side Management system using IoT concepts.

## 🚀 Features
- Smart Meter Dashboard
- Dynamic Electricity Pricing
- Load Monitoring & Control
- Automatic Device Shutdown on Overload
- Cost Calculation
- Power Usage Graph
- Load Prediction

## 🛠️ Technologies Used
- Python
- Streamlit
- MQTT

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python -m streamlit run app.py

## 🎯 Outcome
Efficient energy management using smart grid technology.
